# .github



## Install

```bash
sudo apt install -y asciinema
```



## Record

```bash
# Press ctrl+d to exit
asciinema rec demo.cast
```



## Play

```bash
asciinema play demo.cast
```



## Share

```bash
asciinema auth
asciinema upload demo.cast
```

- [asciinema](https://docs.asciinema.org/manual/cli/quick-start/)
